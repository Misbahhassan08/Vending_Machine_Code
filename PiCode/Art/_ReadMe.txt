Image names must be in the format:
*** {ImageName} must not contain _
{ImageName}_{ImageWidth}x{ImageHeight}_COMPRESSED_RGBA_ASTC_{CompressionType}_KHR.raw

Button names must in the same format as Image names with one difference:
{ImageName} must end with On or Off
On for the selected state
Off for the un-selected state

Font Names must be in the format:
*** {FontName} must not contain _
{FontName}_{FontSize}_ASTC.glyph
xfont data must match font name and end in .xfont
